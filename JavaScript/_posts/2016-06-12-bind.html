既然提了EcmaScript那还是备注下地址好：http://www.ecmascript.org

贴上ES5：http://www.ecma-international.org/publications/files/ECMA-ST/Ecma-262.pdf

19.2.3.2 Function.prototype.bind ( thisArg , ...args)

一：绑定this：

		栗子：
		var object = {x: 1};
		function add(val){
			return this.x + val;
		}
		var objectAdd = add.bind(object);  //这里便是了~~~

二：currying(柯里化)：

		栗子：
		function fun(a, b, c){}
		var fun_ab = fun.bind(undefined, 'value of a', 'value of b');
		fun_ab('value of c');
